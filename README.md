## Keyboard shortcuts

- `s` => skip the current song
- `l` => toggle light theme
- `v` => toggle visualizer
- `m` => increment tux (the penguin) mode
- `n` => toggle nyan doge mode
- `p` => toggle party mode
- `up arrow` => increment audio volume
- `down arrow` => decrement  audio volume
